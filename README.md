# 🚀 Tauri Templates

A curated collection of modern, production-ready Tauri templates designed for speed, aesthetics, and developer experience.

## ✨ Available Templates

All templates share a modern, production-ready foundation:

- **Runtime:** [Tauri 2.0](https://tauri.app/)
- **Frontend:** React 19 + TypeScript + [Vite 7](https://vitejs.dev/)
- **Routing:** [React Router 7](https://reactrouter.com/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/) + [Radix UI](https://www.radix-ui.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **State:** [Zustand](https://zustand-demo.pmnd.rs/)

---

### ⚡ [tauri-vite-react-surrealdb](./tauri-vite-react-surrealdb)

A high-performance foundation for Tauri applications using SurrealDB WASM for ultra-fast local data persistence and a sleek UI.

- **Unique Feature:** [SurrealDB WASM](https://surrealdb.com/docs/surrealdb/integration/libraries/wasm) integration for zero-latency local operations.
- **Theme:** [next-themes](https://github.com/pacocoursey/next-themes) support for seamless Dark/Light modes.

### 🚀 [tauri-surrealdb-webcontainer](./tauri-surrealdb-webcontainer)

A professional-grade, IDE-like template with an embedded terminal and WebContainer support. Perfect for building code editors or browser-based developer tools within Tauri.

- **Unique Features:** [WebContainer API](https://webcontainers.io/), [xterm.js](https://xtermjs.org/) terminal, and [React Resizable Panels](https://github.com/bvaughn/react-resizable-panels).
- **Complexity:** Higher - ideal for advanced developer experience (DX) tools.

## 🛠️ Quick Start

1. **Clone the repository:**

   ```bash
   git clone https://github.com/roscoevanderboom/tauri-templates.git
   ```

2. **Navigate to a template (e.g., tauri-vite-react-surrealdb):**

   ```bash
   cd tauri-vite-react-surrealdb
   ```

3. **Install dependencies:**

   ```bash
   pnpm install
   ```

4. **Run the application:**
   ```bash
   pnpm tauri dev
   ```
